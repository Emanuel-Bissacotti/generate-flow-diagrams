Bem-vindo ao Gerador de Diagramas de Atividades

O que é o Gerador de Diagramas de Atividades?

O gerador de diagrama de atividades é um projeto que visa gerar um diagrama de atividades com base nas informações dadas ao usuário.

Vamos Começar?

Insira as informações na página de formulário para gerar o diagrama ou olhe um exemplo na página de Exemplo de Diagrama de Atividades.

Como funciona o Gerador de Diagrama de Atividades?

O Gerador de Diagrama de Atividades primeiramente recebe as informações do usuário pelo site descritas por ele, depois as informações são recebidas em um banco de dados, que envia para uma inteligência artificial que envia de volta para o banco de dados o código para gerar o Diagrama. O banco de dados depois envia o código para o programa Plant UML que gera o diagrama.
